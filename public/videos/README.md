# Video Files for Tourism Page
Please add your promotional video file as:
- `/public/videos/ivory-coast-promo.mp4`
- `/public/videos/ivory-coast-promo.webm` (optional, for better browser support)
## Recommended Video Specifications:
- Format: MP4 (H.264 codec)
- Resolution: 1920x1080 or higher
- Duration: 30-60 seconds
- File size: Under 50MB for optimal loading
## Current Status:
The page will display a beautiful gradient fallback with Ivory Coast flag and messaging until the video is added. The play/pause button and all animations will work once the video is in place.
